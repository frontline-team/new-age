# A creative design portfolio

## Project setup

```
git clone https://github.com/frontline-team/new-age.git
cd new-age
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your tests

```
npm run test
```

### Lints and fixes files

```
npm run lint
```
